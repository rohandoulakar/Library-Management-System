# Library Management System

A full-stack **Library Management System** built with **Node.js** and **Express**, featuring RESTful APIs, real-time server responses, and **Cypress end-to-end testing**. This project allows users to manage books, users, and borrow/return workflows efficiently.

---

## Features
- **CRUD Operations:** Add, update, delete, and view books and users.
- **RESTful APIs:** Structured endpoints for library resource management.
- **Automated Testing:** Cypress used for end-to-end testing of core workflows.
- **Server Validation:** Input validation and error handling to ensure data integrity.
- **Modular Architecture:** Scalable and maintainable backend structure.

---

## Prerequisites
- **Node.js** v16+
- **npm** package manager

---

## Setup Instructions
1. Install dependencies: npm install

## Running the Application
1. Start the server: npm start
2. Open browser: http://localhost:3000

## Running Cypress Tests
### Headless Mode
npm run cy:run

To run the Cypress tests make sure to start the server
